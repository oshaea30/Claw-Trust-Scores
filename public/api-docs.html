API Docs

Base URL: https://claw-trust-scores-production.up.railway.app

Auth

Send x-api-key on protected endpoints.

Core Endpoints

POST /v1/events log trust events

GET /v1/score?agentId=... read trust score

POST /v1/integrations/clawcredit/preflight policy decision

GET /v1/usage limits and monthly usage

GET /v1/audit/decisions?format=json|csv export decision logs

GET /v1/plans public plan limits

GET /health service status

Quick Example

curl "https://claw-trust-scores-production.up.railway.app/v1/score?agentId=agent:demo:vendor-1" \
  -H "x-api-key: YOUR_KEY"
