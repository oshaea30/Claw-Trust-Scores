Privacy Policy

Last updated: February 19, 2026

Claw Trust Scores stores only the data needed to deliver trust scoring and policy decisions for your account.

Data We Process

  • Account email and API key metadata
  • Events you submit (agent IDs, event types, optional details)
  • Derived trust scores and decision logs
  • Webhook configuration and delivery logs

How We Use Data

  • Compute trust scores and preflight decisions
  • Enforce plan limits and rate controls
  • Support audit exports and troubleshooting

Retention

Data is retained while your key is active. If you revoke a dynamic API key, account-scoped data tied to that key is removed from active storage.
